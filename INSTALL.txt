This file contains instructions on installing and interacting with the current blackjack game. 

Similar to 3110 assignment, once you have downloaded and unzipped the source code from CMS, run "make clean" and "make build". 

After this, run make play. All the functionality is not complete yet. When the terminal prompts you for a file, enter 'init.json'. 

After this you can experiment with some of the hit/stay functionality that we demoed.
